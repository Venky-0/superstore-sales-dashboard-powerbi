# superstore-sales-dashboard-powerbi
Interactive Sales &amp; Profit Analysis Dashboard using Power BI on Superstore Dataset

Project Overview

This project is an interactive data visualization dashboard developed using Power BI on the Superstore dataset to analyze sales performance, profit trends, and business insights.
The dashboard provides interactive visualizations to help stakeholders understand sales performance, profitability, discount impact, and regional trends for better decision-making.

Key Features

KPI Metrics
1. Total Sales
2. Total Profit
3. Total Quantity Sold
4. Average Discount

Visualizations
1. Regional Sales Performance
2. Profit by Category
3. Sales by Sub-Category
4. Discount vs Profit Analysis

Interactive Filters
1. Region
2. Category
3. Segment

Tools Used

1. Power BI
2. Sample Superstore Dataset (CSV/Excel)

Key Insights

1. West region generated strong sales performance.
2. Technology category contributed higher profits.
3. Discounts impacted overall profitability.
4. Product sub-categories showed varying sales trends.
